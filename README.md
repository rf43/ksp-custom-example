# ksp-custom-example
An example of a custom KSP implementation using Kotlin

## How to run
1. Clone the repository
2. Navigate to the root directory of the project
3. Run the following command in the root directory of the project:
```shell
./gradlew run
```

## How to test
1. Clone the repository
2. Navigate to the root directory of the project
3. Run the following command in the root directory of the project:
```shell
./gradlew test
```
